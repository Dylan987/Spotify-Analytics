Hi everyone. This web app was made by Dylan Snelgrove and Evelyn Li. It allows users to analyze their favorite songs, artists, albums, and playlists. Additionally, users can create custom playlists based on selected "seeds" and attributes.
This website uses the Spotify API, and we thank Spotify for making it available for our use.
In making this website, we practiced web development, developed teamwork skills, and learned AJAX and RESTful API.
